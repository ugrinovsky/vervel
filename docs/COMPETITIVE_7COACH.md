# Vervel и 7Coach — конкурентный анализ

**Назначение:** внутренний срез для продуктовых и продажных решений.
**Дата:** май 2026.
**Источник по 7Coach:** публичный сайт 7coach.pro (апрель–май 2026); то, что не заявлено публично, помечено «не заявлено».
**Источник по Vervel:** код репозитория (`apps/api`, `apps/web`).

---

## Позиционирование

| | **7Coach** | **Vervel** |
|---|---|---|
| Суть | «Цифровой блокнот тренера»: клиенты, расписание, деньги, абонементы | Платформа **тренер + атлет**: планирование, нагрузка, коммуникации, ИИ |
| Основной пользователь | Тренер | **Два кабинета** — тренер и атлет равноправны |
| Канал | Нативные iOS / Android + сайт | Web PWA (React + Vite); нативных сторов нет |

---

## Сравнение по областям

| Область | 7Coach | Vervel |
|---------|--------|--------|
| Карточка клиента | Фото, контакты, заметки; теги в Premium | CRM-поля на `TrainerAthlete` (crmStatus, crmNote, nextFollowUpAt) + лиды `TrainerLead` с воронкой |
| Расписание | Слоты, конфликты, стоимость слота, отметка оплаты | `ScheduledWorkout` с date/time, assignedTo (атлет / группа), статус (scheduled/completed/cancelled/skipped) |
| Повторяющиеся занятия | Повторяющиеся слоты | `WorkoutTemplate` + шаблоны в форме создания |
| Группы | Групповые занятия, посещаемость | `TrainerGroup` + pivot `group_athletes`; fan-out дневников через `ScheduledWorkoutFanoutService` |
| **Абонементы** | Учёт пакета, остаток занятий, напоминания о продлении | **Не реализовано** (планируется, см. MEMBERSHIPS_INTEGRATION_PLAN.md) |
| Финансы тренера | Доход/расход, прибыль по типам занятий, периоды | Кошелёк только для ИИ-операций (`balance_transactions`); P&L практики тренера **не реализован** |
| Аналитика | Клиенты, сессии, финансы | У атлета: ACWR, CTL/ATL/TSB, зоны; у тренера: воронка CRM, copilot priority list |
| Содержание тренировки | Не позиционируется как конструктор программы | Каталог упражнений, зоны/мышцы, кастомные упражнения (`TrainerCustomExercise`), `WorkoutDraft` |
| Коммуникации | Не в центре продукта | Личные и групповые чаты (`Chat`/`Message`), GIF Klipy, видеозвонки LiveKit (1:1 и группа) |
| ИИ | Не заявлено в том же объёме | OCR-распознавание, парсинг текста, генерация, чат-ассистент, Trainer/Athlete Copilot |
| Мотивация атлета | Не заявлено | Стрики (`UserStreak`), достижения (`Achievement`), XP, лидерборд группы |
| Дневник атлета | Не в продукте | `Workout` с RPE, зонами, объёмом; атлет ведёт самостоятельно |
| Офлайн | Упоминается в листингах сторов | PWA; полный офлайн не задекларирован |
| Авторизация | Сторы | Email/пароль, VK OAuth, Яндекс OAuth |
| Монетизация | Бесплатно до ~7 клиентов; Premium ~$5/мес или ~$40/год | Баланс для ИИ (ЮKassa); подписка тренера — планируется |
| B2B / студии | Кастомные решения по запросу | Не выделено отдельным предложением |

## Где Vervel объективно сильнее

1. **Атлет — полноправный пользователь.** Дневник, нагрузка, план в одном месте; не просто «запись в блокноте тренера».
2. **Глубина тренировочного процесса.** Периодизация, ACWR, зоны, RPE, кастомные упражнения.
3. **Коммуникации внутри продукта.** Чат, групповой чат, видеозвонки — не нужны сторонние мессенджеры.
4. **ИИ-инструменты.** Ввод с фото/текста, копилоты — дифференциатор при готовности аудитории платить.
5. **CRM-воронка с лидами.** Рост практики тренера шире, чем просто календарь действующих клиентов.
6. **Удержание атлета.** Стрики, ачивки, XP, лидерборды.
7. **РФ-контур.** ЮKassa, VK, Яндекс.
8. **Скорость итераций.** Один web-клиент — деплой без ревью сторов.

## Где 7Coach сильнее или проще заходит

1. **Простая история продаж.** «Клиенты + расписание + деньги» — понятно без объяснений ИИ и дневников.
2. **Нативные приложения** в App Store / Google Play — доверие и привычная установка.
3. **Финансы практики.** Доход/расход, прибыль по типам занятий — у Vervel этого нет.
4. **Абонементы** — простой и востребованный инструмент; у Vervel пока отсутствует.
5. **Низкий бесплатный порог.** До ~7 клиентов без оплаты — меньше трения при онбординге.
6. **B2B клубы** — явное предложение для студий.
7. **Предсказуемая цена.** ~$5/мес понятнее, чем «баланс за ИИ» для нетехнической аудитории.

## Сегменты

| Сегмент | Кому логичнее |
|---------|---------------|
| «Только учёт клиентов, расписания и кассы» | 7Coach |
| «Хочу вести программу, видеть нагрузку атлета, общаться и использовать ИИ» | Vervel |
| Студии / несколько тренеров без кастома | 7Coach (у Vervel нет multi-trainer B2B) |
| Тренер с техкомпетентными атлетами (кроссфит, функциональные виды) | Vervel |

---

## Приоритетные пробелы для закрытия

По порядку важности для конкурентного паритета:

1. **Абонементы** — самый частый запрос; закрывает главный аргумент «у 7Coach есть, у вас нет» (план: MEMBERSHIPS_INTEGRATION_PLAN.md)
2. **Финансы практики** — доход/расход тренера; пока не в роудмапе
3. **Нативные приложения** — дорого, откладывается; PWA + web push — рабочий паллиатив
